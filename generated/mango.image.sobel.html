mango.image.sobel¶

mango.image.sobel(input, output=None, mode='mirror', cval=None)¶

Sobel gradient-magnitude filter.

Parameters:
  • input (mango.Dds) – mango.Dds image which is to be Sobel filtered.
  • mode (str) – String indicating how to handle borders, one of “mirror”, “constant”, “reflect”.
  • cval (scalar) – When mode=='constant' use this value to initialise borders.
Return type:

mango.Dds
Returns:

Sobel filtered gradient-magnitude image.
