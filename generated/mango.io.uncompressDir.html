mango.io.uncompressDir¶

mango.io.uncompressDir(zipDirPath, dest=None, preserve=True, mpiComm=<mpi4py.MPI.Intracomm object at 0x7f72206bdb70>)¶

Uncompress files in a specified directory. Can take advantage of MPI by uncompressing files per MPI process.

Parameters:
  • zipDirPath (str) – Directory containing compressed files.
  • dest (str) – Directory where uncompressed files are written (must exist). If dest==None, files are uncompressed in directory zipDirPath.
  • preserve (bool) – If False, original compressed files are deleted after uncompressed file is created.
  • mpiComm (mpi4py.MPI.Comm) – Communicator for MPI parallelism.
Return type:

list of str
Returns:

List of the uncompressed file names.
