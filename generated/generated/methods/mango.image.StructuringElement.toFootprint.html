mango.image.StructuringElement.toFootprint¶

StructuringElement.toFootprint((_StructuringElement_3)arg1) → object :¶

Returns a numpy.array of bool elements. The True values are the structuring elements. Structuring element offsets are relative to the centre element of the array (centre index footprint.shape//2).

Return type:numpy.array of bool
Returns:Footprint array of this structuring element.
