mango.image.SphericalHistogram.addSamples¶

SphericalHistogram.addSamples((_SphericalHistogram_float64)arg1, (object)arg2, (object)arg3) → None :¶

Increments the triangular bin which intersects the ray (extending from self.sphere_c through coords[i]) by counts[i].

Parameters:
  • coords ((n,3) shaped numpy.array) – A (n,3) shaped array of coordinates, ray-intersecting bins are incremented by the corresponding counts element.
  • count – A (n,) shaped array of bin-increment elements.
