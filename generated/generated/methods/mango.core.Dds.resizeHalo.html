mango.core.Dds.resizeHalo¶

Dds.resizeHalo((_Dds_uint16)arg1, (object)halo[, (bool)serializePerNode=False]) → None :¶

Resizes the halo/border region of the halo-sub-domain. This is somewhat dangerous, as it deletes the underlying C++ array, so any existing references to the original array (i.e. numpy.ndarray reference returned by asarray()) will refer to deleted memory with the potential for segmentation-fault.

Parameters:
  • halo (int or 3-sequence) – Halo resized to this value.
  • serializePerNode (bool) – If True, the re-allocation of the sub-domain C++ array is performed in serial on each node. This avoids the simultaneous duplication of the entire array (per node) at the expense of MPI blocking/barrier.
C++ signature :
void resizeHalo(DistributedDataStruct<unsigned short> {lvalue},boost::python::api::object [,bool=False])
