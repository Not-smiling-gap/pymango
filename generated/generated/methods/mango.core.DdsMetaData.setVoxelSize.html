mango.core.DdsMetaData.setVoxelSize¶

DdsMetaData.setVoxelSize((_MetaData_Dds_uint16)arg1, (object)size[, (object)unitstr=None]) → None :¶

Set the side lengths of the rectangular-prism voxels.

Parameters:
  • size (3-sequence) – (zsz,ysz,xsz) voxel size Dds array elements.
  • unitstr (str) – The unit string, e.g. 'millimetre' for voxel side-length. If None, the size unit is left unaltered.
