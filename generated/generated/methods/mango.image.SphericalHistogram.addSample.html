mango.image.SphericalHistogram.addSample¶

SphericalHistogram.addSample((_SphericalHistogram_float64)arg1, (object)coord[, (float)count=1]) → None :¶

Increments the triangular bin which intersects the ray (extending from self.sphere_c through coord) by count.

Parameters:
  • coord (3 sequence) – Increment the bin which intersects the ray passing through this coordinate.
  • count (float) – Increment the intersecting bin by this amount.
