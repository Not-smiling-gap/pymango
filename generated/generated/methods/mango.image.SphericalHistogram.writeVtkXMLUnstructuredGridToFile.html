mango.image.SphericalHistogram.writeVtkXMLUnstructuredGridToFile¶

SphericalHistogram.writeVtkXMLUnstructuredGridToFile((_SphericalHistogram_float64)arg1, (str)filename[, (object)comm=None[, (int)root=0]]) → None :¶
Writes XML file containing vtkUnstructuredGrid representation of spherical histogram
(requires mango compiled with VTK support, i.e. mango.haveVTK == True).
Parameters:
  • filename (str) – Path of file to be written with vtkUnstructuredGrid data.
  • comm (mpi4py.MPI.Comm) – MPI communicator used to write the data (only MPI process with comm.Get_rank() == root writes data to file).
  • root (int) – MPI communicator which writes the data to file. Data is not MPI-reduced by this method.
