mango.unittest.TestCase.createTmpDir¶

TestCase.createTmpDir(dirName=None, mpiComm=<mpi4py.MPI.Intracomm object at 0x7f72206bdb70>, rootRank=0)[source]¶

Creates unit-test temporary directory. Directory and files are cleaned up (removed) on python exit.

Parameters:
  • dirName (str) – Pathname (absolute or relative to the root temporary test directory) to be removed. If absolute, must be a sub-directory/descendant of the root unit-test directory.
  • mpiComm (mpi4py.MPI.Comm) – MPI communicator object, blocking (barrier) is performed using this object (to avoid an MPI process racing to access the yet-to-be-created directory).
  • rootRank (int) – Rank of the root process which creates the directory.
Return type:

str
Returns:

Full path of newly created directory.
