mango.io.compressDir¶

mango.io.compressDir(dirPath, dest=None, method='application/x-bzip2', preserve=True, mpiComm=<mpi4py.MPI.Intracomm object at 0x7f72206bdb70>)¶

Compress files in a specified directory. Can take advantage of MPI by compressing files per MPI process.

Parameters:
  • dirPath (str) – Directory containing files which are to be compressed.
  • dest (str) – Directory where compressed files are written (must exist). If dest==None, files are compressed in directory dirPath.
  • preserve (bool) – If False, original uncompressed file is deleted after compressed file is created.
  • mpiComm (mpi4py.MPI.Comm) – Communicator for MPI parallelism.
Return type:

list of str
Returns:

List of the compressed file names.
