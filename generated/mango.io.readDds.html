mango.io.readDds¶

mango.io.readDds(fileName, halo=(0, 0, 0), mpidims=(0, 0, 0))¶

Reads mango netCDF data from specified file/directory.

Parameters:
  • fileName (str) – The netCDF file/directory from which 3D array data is read.
  • halo (3-sequence) – Extra number of halo/overlap voxels (nZ,nY,nX) on the read array.
  • mpidims (3-sequence) – The MPI cartesian processor layout (nZ,nY,nX) for the returned mango.Dds object.
Return type:

mango.Dds
Returns:

mango.Dds object initialised with data from file.
