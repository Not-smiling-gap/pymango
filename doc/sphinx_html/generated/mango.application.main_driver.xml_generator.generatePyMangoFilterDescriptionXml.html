mango.application.main_driver.xml_generator.generatePyMangoFilterDescriptionXml¶

mango.application.main_driver.xml_generator.generatePyMangoFilterDescriptionXml(xmlStringPart1=None, xmlStringPart2=None, filterClsList=None)[source]¶

Generate the XML filter description string (e.g. the pymango.xml string) associated with the pymango executable. Resulting XML string suitable for qmango parsing.

Parameters:
  • xmlStringPart1 (str) – Prefix XML string prepended to the individual filter XML descriptions.
  • xmlStringPart2 (str) – Suffix XML string appended to the individual filter XML descriptions.
  • filerClsList (sequence of mango.application.main_driver.MainDriverFilter class objects.) – Generate (and validate) XML for this list of classes. If None, generates the class list using the mango.application.main_driver.getAllRegisteredMainDriverFilterClasses function.
Returns:

String of XML describing the main-driver filters in the given filterClsList sequence.
