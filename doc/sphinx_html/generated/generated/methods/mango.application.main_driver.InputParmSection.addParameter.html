mango.application.main_driver.InputParmSection.addParameter¶

InputParmSection.addParameter((InputParmSection)arg1, (str)name, (object)value[, (bool)canOverwrite=True]) → None :¶

Adds the (name, str(value)) parameter pair to this parameter set.

Parameters:
  • name (str) – Name of parameter to add to this parameter set.
  • value (object) – Value to be associated with parameter named name.
  • canOverwrite (bool) – If True, this parameter can have a different value assigned.
