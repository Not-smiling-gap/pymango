mango.Dds.asarray¶

Dds.asarray((object)arg1) → object :¶

Returns numpy.array object which references the elements of this Dds object. This is equivalent to self.subd_h.asarray(), i.e. returns the halo inclusive array.

Return type:numpy.array
Returns:numpy.array object which references the elements of this Dds object.
Example usage::
>>> dds = mango.zeros(shape=(10,50,50), dtype='uint16')
>>> print(dds[0,0,0])
0
>>> a = dds.asarray()
>>> a += 16
>>> print(dds[0,0,0])
16
