mango.Dds.updateHaloRegions¶

Dds.updateHaloRegions((_Dds_uint16)arg1) → None :¶

Updates halo voxels to be consistant with neighbouring MPI sub-domains. That is, reads from the ‘exclusive’ MPI-sub-domains of each process, and then writes that data into the overlap regions of the neighbouring MPI processes, thus updating the overlap/halo regions with data from the interior regions.
