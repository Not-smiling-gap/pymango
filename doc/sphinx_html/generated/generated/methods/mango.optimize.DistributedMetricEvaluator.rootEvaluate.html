mango.optimize.DistributedMetricEvaluator.rootEvaluate¶

DistributedMetricEvaluator.rootEvaluate(x)¶

Broadcasts x to all processes and then does local evaluation (self.evaluate(x)). MPI reduces (op=mpi.SUM) the results from all MPI processes and returns the reduced result. Should only be called by the self.root-rank process.

Parameters:x (broadcastable object) – Function parameter.
Returns:Reduction of the evaluate() values from MPI processes.
