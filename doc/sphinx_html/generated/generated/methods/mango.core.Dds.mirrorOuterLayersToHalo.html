mango.core.Dds.mirrorOuterLayersToHalo¶

Dds.mirrorOuterLayersToHalo((_Dds_uint16)arg1, (bool)copyFirstLayer) → None :¶

Mirrors the layers closest to the global-boundary into the halo-regions. It treats the border as a mirror. If copyFirstLayer=False, a data set looking like: x x x | 1 2 3 4 (‘|’’ being boundary) will end up looking like: 4 3 2 | 1 2 3 4. If copyFirstLayer=True it will copy the first layer (duplicating it), so a data set looking like: x x x | 1 2 3 4 (‘|’ being boundary) will end up looking like: 3 2 1 | 1 2 3 4 (note layer 1 being duplicated).

Parameters:copyFirstLayer (bool) – If True, copy first global-border-layer of data set.
