mango.image.SphericalHistogram.getBinCentre¶

SphericalHistogram.getBinCentre((_SphericalHistogram_float64)arg1, (object)coord) → object :¶

Returns the centre of the triangular bin that intersects the ray which extends from the histogram sphere centre through the specified coordinate.

Parameters:coord (3 sequence) – Coordinate for ray used to find triangular bin of intersection.
Return type:3 sequence
Returns:The centre of the intersected triangular bin.
