mango.image.SphericalHistogram.asVtkUnstructuredGrid¶

SphericalHistogram.asVtkUnstructuredGrid((_SphericalHistogram_float64)arg1) → object :¶
Returns vtk.vtkUnstructuredGrid of spherical histogram.
Requires mango compiled with VTK support, (i.e. mango.haveVTK == True) and requires the vtk python module available for import.
Return type:vtk.vtkUnstructuredGrid
Returns:A vtk.vtkUnstructuredGrid with triangular cells and associated cell data: 'facet_count', 'facet_index', 'facet_area', 'facet_count_div_area', etc.
