mango.application.main_driver.xml_generator.generateXmlFromCommandLineArgs¶

mango.application.main_driver.xml_generator.generateXmlFromCommandLineArgs(args)[source]¶

Main routine used to generate the '<install_prefix>/share/pymango.xml' XML file which describes main-driver filters and data-types. XML description are generated for filter classes which are registered with the mango.application.main_driver module.

The args argument is required to have the following attributes:

args.loggingLevel (str)
The logging level string, one of 'INFO', 'DEBUG', 'WARN', 'ERROR', 'CRITICAL', 'NONE'.
args.xmlPart1 (str)
XML string pre-pended to individual filter XML descriptions.
args.xmlPart2 (str)
XML string appended after the individual filter XML descriptions.
args.outputFileName (str)
Name of the file in which the generated XML string is written.
Parameters:args (object) – Parameter object, see above.
