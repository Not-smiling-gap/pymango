mango.io.uncompress¶

mango.io.uncompress(zipFilePath, dest=None, preserve=True)¶

Uncompress a specified file.

Parameters:
  • zipFilePath (str) – File to be uncompressed. Must have extension which indicates the compression method (i.e. extension is '.bz2' or '.gz').
  • dest (str) – Name of uncompressed file If dest==None, file name is zipFilePath minus the extension related to the compression method.
  • preserve (bool) – If False, original uncompressed file is deleted after compressed file is created.
