mango.core.sum¶

mango.core.sum(input, dtype=None)¶

Returns sum of all non-masked Dds elements.

Parameters:
  • input (Dds) – Input elements for summation is calculated.
  • dtype (numpy.dtype or dtype str) – Type used for summation of elements.
Return type:

scalar
Returns:

Sum of all elements.
