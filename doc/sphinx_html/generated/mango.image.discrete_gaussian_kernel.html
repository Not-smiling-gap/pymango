mango.image.discrete_gaussian_kernel¶

mango.image.discrete_gaussian_kernel(sigma=1.0, errtol=0.01, dim=3)¶

Discrete-Gaussian convolution kernel.

Parameters:
  • sigma (float) – The Discrete-Gaussian standard deviation.
  • errtol (float) – 1D Error tolerance which determines the size of the kernel window, smaller value gives larger kernel window.
  • dim (int) – Spatial dimension of kernel.
Return type:

numpy.array
Returns:

A dim dimensional Discrete Gaussian kernel.
